## 🛑 El Error del Junior

"Depende de los datos"—obvio. No tiene criterios específicos para cada tipo de visualización.

## 🧠 El Enfoque Curse Dev (Contra-pregunta)

* **Tú:** *"Chart selection is about matching the visual to the analytical goal. Would it help if I walked through my decision framework?"*

## 💻 La Respuesta Técnica

**Guía para elegir gráficas:**

| Objetivo Analítico | Gráfica Recomendada | Por qué |
|-------------------|---------------------|---------|
| **Comparar categorías** | Barras (verticales/horizontales) | Fácil comparar longitudes |
| **Ver tendencia en tiempo** | Líneas | Muestra continuidad y dirección |
| **Mostrar composición** | Barras apiladas o Treemap | Parte-todo claro |
| **Distribución de datos** | Histograma o Box plot | Muestra spread y outliers |
| **Relación entre variables** | Scatter plot | Correlaciones visibles |
| **Progreso hacia meta** | Bullet chart o Gauge | Target vs actual |

**Errores comunes a evitar:**
- **Pie charts** con más de 3-4 categorías (difícil comparar ángulos)
- **3D effects** (distorsiona percepción de tamaño)
- **Double Y-axis** (puede engañar visualmente)

## 🇺🇸 El Script en Inglés (Para la entrevista)

> *"Choosing the right chart type is about matching the visualization to the analytical question you're trying to answer. I use a mental framework based on intent. If the goal is comparison—like sales by region—I use bar charts because humans are excellent at comparing lengths. For trends over time, line charts are ideal because they show continuity and direction clearly. If I need to show composition—how parts make up a whole—I'll use stacked bars or a treemap, but never a pie chart with more than 3-4 slices because angle comparison is cognitively hard. For distributions, histograms or box plots reveal spread, central tendency, and outliers. For relationships between two variables, scatter plots make correlations visible instantly. I also consider the audience: executives often prefer simple, high-level visuals like bullet charts showing actual vs target. Analysts might want more detail like box plots. The key mistakes I avoid: 3D effects (they distort perception), dual Y-axes (they can mislead), and overusing pie charts. My rule: the chart should make the insight obvious within 3 seconds. If the stakeholder has to study it, I haven't done my job."*

## 🔑 Vocabulario Clave

* **Analytical goal:** Objetivo analítico
* **Comparison:** Comparación
* **Trend:** Tendencia
* **Composition:** Composición
* **Dual Y-axis:** Eje Y dual
